## 📌 Présentation du projet
**StartSmart** est une plateforme web innovante dédiée à l’entrepreneuriat digital. Elle permet aux utilisateurs de proposer des idées de startups, de collaborer avec d’autres profils, et de découvrir des opportunités de financement.

Grâce à l’intégration de fonctionnalités intelligentes, la plateforme facilite la mise en relation entre entrepreneurs et investisseurs, tout en encourageant la création de projets innovants et responsables.

Le projet s’inscrit dans la thématique de l’économie digitale, du futur du travail et du développement durable.

---

## ⚙️ Technologies utilisées

- **Frontend :** HTML5, CSS3  
- **Backend :** PHP  
- **Base de données :** MySQL (via XAMPP)  
- **Architecture :** MVC (Model / View / Controller)

---

## 🛠️ Installation et exécution

1. Installer **XAMPP**
2. Démarrer :
   - Apache
   - MySQL
3. Placer le projet dans le dossier :
htdocs/
4. Importer la base de données dans **phpMyAdmin**
5. Accéder au projet via : http://localhost/nom_du_projet

---

## 🧱 Structure du projet

Le projet suit l’architecture **MVC (Model / View / Controller)** :

- **Model :** gestion des données et base de données  
- **View :** interface utilisateur (HTML/CSS)  
- **Controller :** logique de traitement
