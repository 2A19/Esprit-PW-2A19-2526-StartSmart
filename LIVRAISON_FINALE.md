## 🎯 Cas d'Utilisation

### Utilisateur Start-Uper

```
1. Voir ressources disponibles
2. Demander accès à une ressource
3. Remplir formulaire de demande
4. Soumettre demande
5. Attendre réponse sponsor
6. Voir l'état dans "Mes Demandes"
7. Si acceptée : accès autorisé
8. Si refusée : essayer une autre
```

### Sponsor/Admin

```
1. Ajouter nouvelle ressource
2. Voir liste des ressources
3. Modifier ressource
4. Voir les demandes en attente
5. Consulter détails demande
6. Accepter ou refuser
7. Si acceptée : accès accordé
8. Voir historique demandes
```
